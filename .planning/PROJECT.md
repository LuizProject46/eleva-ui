# Eleva UI

## What This Is

Eleva UI is the front-end application for building and running evaluations. This repository focuses on delivering a clear, predictable, mobile-first experience for evaluators while keeping UI patterns consistent across the product.

## Core Value

Evaluations are easy to complete correctly on the first try.
